UpdateUrl History Parameter Test

This page demonstrates the UpdateUrl function with different history behaviors.

Test Cases:

1. Default Behavior (adds to history)

Clicking this button uses the default behavior and adds an entry to the browser history.

Usage: UpdateUrl(url)

2. Explicit True Parameter (adds to history)

Clicking this button explicitly passes true and adds an entry to the browser history.

Usage: UpdateUrl(url, true)

3. Replace URL without History (false parameter)

Clicking this button passes false and replaces the current URL without adding to history.

Usage: UpdateUrl(url, false)

How to Test:

  1. Use the browser's back button to verify history behavior
  2. Buttons 1 and 2 should allow navigation back after clicking
  3. Button 3 should replace the URL without allowing back navigation to this page
